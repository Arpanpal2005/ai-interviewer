# 🤖 AI Interviewer

An AI-powered interview preparation platform that analyzes your CV, generates personalized interview questions, and evaluates your answers in real time.

---

## 🚀 Features

* 📄 Upload CV (PDF)
* 🧠 AI-generated personalized interview questions
* 🎯 Questions based on your **projects, skills, and experience**
* 💬 Interactive interview mode (one question at a time)
* 📊 Real-time answer evaluation
* 📝 Final performance report with:

  * Score
  * Strengths
  * Weaknesses

---

## 🧠 How It Works

1. Upload your CV
2. AI extracts key information (skills, projects, experience)
3. Generates tailored interview questions
4. Conducts a step-by-step interview
5. Evaluates each answer using an LLM
6. Provides a detailed performance report
